#imagemin-droid
> Minify images for android project.
> Shrink png/jpg files in res directory.

##Requirement
node & npm

##Install
	$ npm install --global imagemin-droid

##Usage

	$ imagemin-droid -h

	  Usage: imagemin-droid android-res-dir out-dir
